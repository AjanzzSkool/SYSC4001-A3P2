marking.cpp is the implementation that doesn't use semaphores and marking2.cpp is the implementation that does use semaphores.

1. Compile marking.cpp: run this command in your Ubuntu terminal: g++ -std=c++17 marking.cpp -o marking
2. Compile marking2.cpp: run this command in your Ubuntu terminal: g++ -std=c++17 marking2.cpp -o marking2

3.Running marking.cpp: run this command in your Ubuntu terminal: ./marking <Number of TAs you would like (has to be >= 2)> exampaths.txt rubric.txt

4.Running marking2.cpp: run this command in your Ubuntu terminal: ./marking2 <Number of TAs you would like(has to be >= 2)> exampaths.txt rubric.txt
